1. [Install and run agentgateway]({{< link-hextra path="/quickstart/mcp" >}}).
2. Confirm agentgateway is up by opening the [agentgateway UI](http://localhost:3000/ui).
3. Use the same MCP endpoint and port that your local config exposes. Common examples include:
   - `http://localhost:3000/sse`
   - `http://localhost:3000/mcp/http`
   - If you run agentgateway on a different host or port, replace `localhost:3000` in the examples accordingly.
