1. [Install and run agentgateway]({{< link-hextra path="/quickstart/mcp" >}}).
2. Confirm agentgateway is up by opening the [agentgateway UI](http://localhost:15000/ui).
3. Use the MCP endpoint `http://localhost:15000/mcp/http` to connect your client to agentgateway. If you run agentgateway on a different host or port, replace `localhost:15000` in the examples accordingly.
